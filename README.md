Additional Keymaps

* `cmd + shift + p` -> Project window
* `cmd + shift + b` -> Show Branches
* `ctr + shift + a` -> Show context menu
* `ctr + F4 ` -> Close Others
* `ctr + e` -> Export Settings
* `ctr + k` -> Key Promoter
* `ctr + r` -> REPL
* `cmd + q` -> Close Project 
* `ctr + alt + down` -> Jump to next method
* `ctr + alt + up` -> Jump to previous method
